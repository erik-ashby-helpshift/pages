# IMPORTANT - THIS REPOSITORY IS ACCESSED BY AI
This repository is a shared space for collaboration between the Product Research team and AI agents. It contains product research documents intended for AI access.

# Important Guidelines
## Rules for AI Access

*Virtual Container Requirement*: To prevent access to local resources, all AI agents must run within a virtual container.
Content and Data Security

- AI Visibility: Anything contained in this repository should be understood as potentially readable by an AI agent.
- No Sensitive Data: To ensure security, no sensitive data can be stored in this repository.
- Change Tracking: Github is being used as a control so that all content changes (including the addition of new content) can be tracked to the source to prevent the accidental inclusion of sensitive data.
